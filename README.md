# Scratch AI

A machine learning and AI playground built from scratch.

## Overview

This project implements various machine learning algorithms and concepts from scratch using Python, focusing on understanding the fundamentals without relying on high-level ML frameworks.

## Features

- Core ML algorithms implemented from first principles
- Educational notebooks demonstrating key concepts
- Clean, readable code for learning and experimentation

## Getting Started

Explore the notebooks to see implementations and examples of various ML concepts.

## Purpose

Built to understand AI and machine learning at a fundamental level - learning by building everything from scratch.
